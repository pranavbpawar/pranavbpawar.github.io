# Dracky's Portfolio Website

A modern, responsive personal portfolio website showcasing my work as a Security Researcher, Python Developer, and Blockchain enthusiast. This website serves as a comprehensive platform to share my professional journey, skills, and projects.

## Features

- 🎨 Clean and modern UI design
- 📱 Fully responsive layout for all devices
- 🔍 Dynamic search functionality for projects and skills
- 📑 Multiple sections including:
  - Home page with hero section
  - Detailed about page
  - Skills showcase
  - Projects portfolio
  - Contact information
- ⚡ Smooth animations and transitions
- 🔎 Interactive search functionality
- 🎯 Font Awesome integration for modern icons

## Pages

- `index.html` - Main landing page with hero section and key highlights
- `about.html` - Detailed information about my background and achievements
- `projects.html` - Showcase of my technical projects and contributions

## Technologies Used

- HTML5
- CSS3 (with modern features and animations)
- JavaScript (Vanilla)
- Font Awesome for icons
- Modern CSS Grid and Flexbox for layouts

## Project Structure

```
├── index.html          # Home page
├── about.html          # About page
├── projects.html       # Projects showcase
├── styles.css         # Main stylesheet
├── script.js          # Main JavaScript functionality
├── search.js          # Search implementation
├── contact.js         # Contact form handling
└── images/            # Image assets
```


## Customization

### Styling
The website uses a carefully chosen color scheme and typography that can be modified in `styles.css`. Key sections are well-commented for easy customization.

### Content
- Update personal information in the HTML files
- Add new projects in `projects.html`
- Modify skills and achievements in relevant sections
- Add new images to the `images/` directory
